Mit der WOL-Funktion (Wake-On-LAN) können Sie einen Computer von einem beliebigen Gerät im Netzwerk aufwecken.

Einrichten der Funktion

Der Router kann ein "Magic Packet" senden, um Ihren Computer aus dem Ruhemodus/Standby-Modus aufzuwecken.

Stellen Sie sicher, dass Ihr Computer oder Netzwerkadapter die WOL-Funktion unterstützt.

Einstellungen auf der Client-Seite

Wenn Sie die ASUSWRT WOL-Funktion verwenden müssen, stellen Sie bitte sicher, dass Sie folgende Schritte abgeschlossen haben:

Öffnen Sie den Geräte-Manager -> Wählen Sie Ihren Netzwerkadapter aus und klicken Sie auf die rechte Schaltfläche -> Eigenschaften und wählen Energieverwaltung aus hier klicken Sie auf Gerät kann den Computer aus dem Ruhezustand aktivieren und Nur Magic Packet kann Computer aus dem Ruhezustand aktivieren.

Einstellungen auf der Router-Seite

  1. Öffnen Sie einen Webbrowser und navigieren Sie zu WebGUI (http://router.asus.com).


    Hinweis: Weitere Informationen finden Sie unter Wie gelangt man auf die Weboberfläche des Router?
     
  2. Bitte gehen Sie zu Network Tools / Wake on LAN. Geben Sie die MAC-Adresse ein und klicken Sie auf die Schaltfläche Wake up, um Ihren Computer einzuschalten.


    Sie können Ihr Gerät auch in der Offline-Liste speichern. Wenn das Gerät offline ist, klicken Sie einfach auf MAC-Adresse,  um es zum Aufwachbereich hinzuzufügen, dann klicken Sie auf die Schaltfläche Wake up,  um es einzuschalten.
